# Fixtures — reproducible offline grading

Everything here lets anyone reproduce a grade with **zero API calls**.

- `cards/<id>/front.jpg` (+ optional `back.jpg`, `angle.jpg`) — holdout card images.
- `ground-truth.json` — known TAG grades + subgrades per card. The eval scores against this.
- `cache.sqlite` — recorded AI responses keyed by `hash(request) = sha256(model + prompt +
  version + rendered text + images)`. The eval reads from here; the proxy writes here in
  production. A row written by the live proxy is directly reusable as a fixture.
- `snapshot.json` — committed pipeline output, used by the parity gate to catch silent
  regressions (`npm run parity -w eval`).

## Current contents are SYNTHETIC

The committed cards/responses are generated by `eval/src/seed.ts` so the harness runs
end-to-end today. Replace them with real holdout cards + responses recorded from the live
proxy — the harness and metrics code do not change.

### Regenerate the synthetic demo set
```
npm run seed   -w eval     # draws cards, writes ground-truth.json + cache.sqlite
npm run parity -w eval -- --update
```

### Record REAL fixtures (later)
Run the proxy with a key, grade real cards through the web app once; every call is cached
in the proxy DB. Copy those rows into `fixtures/cache.sqlite` and add the cards +
ground-truth entries. From then on the eval replays them for free.
